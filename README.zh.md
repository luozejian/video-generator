# 模拟视频生成器

一个模拟视频生成工具，支持创建精确大小的测试视频文件。

## 功能特性

### 🎥 两种生成模式
- **真实渲染模式**：生成可播放的视频，使用尾部填充技术精确匹配目标大小
- **快速伪造模式**：瞬间生成指定大小的空 .mp4 文件，仅用于测试上传限制

### 🎛️ 灵活配置
- **目标大小**：以 MB 为单位指定视频大小
- **时长**：设置视频长度（秒）
- **分辨率**：自定义宽度和高度

### 🌐 国际化支持
- 支持中英文双语
- 根据用户偏好自动切换

### ✅ 精确大小匹配
- 自动填充数据以精确匹配目标大小
- 填充状态的视觉反馈

## 快速开始

### 安装

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 预览生产构建

```bash
npm run preview
```

## 使用说明

1. **选择生成模式**：
   - 选择「真实渲染模式」生成可播放视频
   - 选择「快速伪造模式」生成瞬间空文件

2. **配置参数**：
   - 输入目标大小（MB）
   - 设置视频时长（仅真实渲染模式）
   - 调整（仅真实渲染模式）

3. **开始生成**：
   - 点击「开始生成」按钮
   - 等待进度条完成

4. **下载结果**：
   - 生成完成后，查看文件详情
   - 点击「下载文件」保存视频

## 技术栈

- **前端框架**：React 19
- **类型系统**：TypeScript
- **构建工具**：Vite
- **样式框架**：Tailwind CSS
- **图标库**：Lucide React
- **视频生成**：Canvas API & MediaRecorder API

## 项目结构

```
├── src/
│   ├── App.tsx          # 主应用组件
│   ├── main.tsx         # 应用入口
│   └── index.css        # 全局样式（包含 Tailwind 指令）
├── tailwind.config.js   # Tailwind CSS 配置
├── postcss.config.js    # PostCSS 配置
├── tsconfig.json        # TypeScript 配置

```

## 浏览器兼容性

- Chrome/Edge：完全支持
- Firefox：有限支持（某些视频格式可能不可用）
- Safari：有限支持（MediaRecorder API 差异）

## 性能说明

- 真实渲染模式可能对大视频资源消耗较大
- 大于 500MB 的视频可能导致浏览器崩溃
- 建议使用快速伪造模式测试上传限制
